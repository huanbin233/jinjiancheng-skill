# 金渐成投资框架总结

> 基于 `raw/text/A002_s.txt` 至 `raw/text/A379_s.txt` 共 378 篇公众号正文整理。本文是研究辅助，不构成投资建议或自动买卖指令；涉及当前价格、估值、财报和政策时，必须再用最新数据校验。

## 一句话总结

金渐成的核心不是“预测某个点位”，而是：顺着全球资金流向和美元周期，把家庭资产分成进取、稳健、防守三层；用优质美元资产做长期底仓，用低位分批、金字塔加仓、做 T 降成本，用高位减仓把利润转成防守型资产，最终形成低成本/负成本持股和更厚的资产安全垫。

## 核心思想

1. 钱不是人硬赚来的，而是顺势承接来的  
   他反复强调“顺势”和“钱的流向”。在 2021-2022 年回笼资金，等待美联储加息周期和美元潮汐；2022 年下旬到 2023 年上旬转向美元资产；随后在 AI、美股科技股和美元资产周期中获得主要收益。

2. 策略会过期，体系要迭代  
   他的最新文章明确说，策略有效期通常只有几年，市场环境、资金体量、风险承受力变化后，旧策略必须重构。不能把历史买卖节点机械复刻。

3. 投资首先是保值，其次才是增值  
   他不把收益率最大化作为唯一目标，而是强调风险可控、现金流、低成本、仓位安全和家庭财富锚。

4. 大方向上，从人民币资产转向全球美元资产  
   他对国内房产、A 股、人民币资产整体偏谨慎，认为过去几年重要机会在美元资产，包括美股、美债、美元现金、港险、美元计价不动产等。

5. 选资产要“要么第一，要么唯一”  
   个股偏好行业龙头、全球龙头、现金流强、护城河深、不可替代的公司。美股中集中体现为七巨头、台积电、消费龙头、生物医药龙头、伯克希尔等。

## 家庭资产框架

他把家庭资产分成“四个钱包”：

1. 短期家庭开支  
   大约 6-8 个月生活费，占 5-10%。

2. 家庭备用金  
   用于突发支出、保险、应急，占 10-20%。

3. 投资资金  
   用于钱生钱，追求较高收益，占 30-40%。这部分可以承受较大波动。

4. 家庭财富锚  
   用于养老、教育、长期安全，占 30-40%。强调保本、稳健、长期现金流。

这套框架里，真正重要的是财富锚。他把它称为压舱石、阻尼器、安全垫。过去很多中国家庭以房产为锚，现在他逐步转向美元资产和全球配置。

## 美股账户分层

### 1. 进取型账户

定位：追求高收益，承受高波动。

主要资产：

- 纳指 100、标普 500 宽基 ETF
- 七巨头
- 台积电
- AMD、博通等科技链资产

特点：

- 科技股集中度高
- 允许阶段性重仓强势资产
- 通过高位减仓、低位加仓、做 T，把核心持仓做成低成本或负成本
- 当收益过高或单票占比过大时，逐步把利润转入防守型账户

### 2. 稳健型账户

定位：攻守平衡，是家庭投资中轴。

主要资产：

- 纳指 100、标普 500 宽基 ETF
- 科技板块
- 生物医药板块
- 消费龙头板块
- 伯克希尔、VISA 等偏金融/投资类资产

特点：

- 仍以美股优质资产为主，但分散度高于进取型
- 板块结构更均衡
- 既参与科技成长，又靠消费、医药、伯克希尔降低波动

### 3. 防守型/长期账户

定位：压舱石、安全垫，未来占比提升到非常重要的位置。

主要资产：

- 宽基 ETF：SPY、VOO、IVV、QQQ 等
- 美债和债券 ETF：BIL、SHY、TLT 等
- 高股息/防守个股：可口可乐、强生、宝洁、MO 等
- 红利和消费 ETF：SCHD、XLP 等
- 伯克希尔
- 港险、美元定存、不动产等更长期资产

特点：

- 不是为了短期暴利，而是为了活得久、抗波动
- 进取型和稳健型高位减仓后的资金，会逐渐转入这里
- 随着资金体量变大，防守型配置的重要性提升

## 美股选股偏好

他偏好的美股资产可以分为几类：

1. 科技核心  
   英伟达、微软、谷歌、亚马逊、Meta、苹果、特斯拉、台积电、AMD、博通等。

2. 宽基指数  
   标普 500、纳指 100 是长期底仓和不同账户的共同基础。

3. 消费龙头  
   Costco、沃尔玛、麦当劳、宝洁、可口可乐等，偏稳健和防守。

4. 生物医药  
   礼来、强生、联合健康、诺和诺德、默沙东等，但他也会承认该板块波动和个股风险。

5. 金融/投资类压舱石  
   伯克希尔、VISA 等。

他的选股原则不是“便宜就买”，而是先确认资产质量，再看价格和周期位置。

## 操作系统

### 1. 建仓

建仓不是一次性梭哈，而是先判断三件事：

- 资产是否足够优质，是否符合“第一或唯一”
- 当前是否处在可接受的周期和估值位置
- 自己是否有足够现金和仓位应对继续下跌

典型做法：

- 设置多个买入节点
- 跌到节点分批买
- 越跌越买得多，即金字塔加仓
- 不满仓，保留后续弹药
- 如果基本面没有恶化，下跌反而是加仓窗口

### 2. 加仓

加仓来自三类机会：

- 市场恐慌或系统性下跌
- 优质个股被错杀
- 原有逻辑被财报、行业趋势继续验证

加仓纪律：

- 不追高加仓
- 不因短期上涨而情绪化买入
- 用仓位和资金计划约束自己
- 价格越低、确定性越强，买入权重越高

### 3. 做 T 和降成本

做 T 在他的体系里不是短线赌博，而是成本管理工具。

核心思路：

- 保留 60-70% 底仓，防止踏空
- 用 30-40% 仓位做小波段
- 大跌后买入，反弹后卖出部分
- 目标是降低成本、腾出资金、扩大安全边际

如果做 T 能把核心资产变成低成本或负成本，后续持有心态会更稳。

### 4. 减仓

减仓不是看空资产，而是管理风险。

触发条件：

- 个股或指数到达预设高位节点
- 市场情绪过热
- 单票占比过高
- 已经实现较大浮盈，需要落袋部分收益
- 需要把进取型利润转入防守型账户
- 基本面、竞争格局或原始逻辑变坏

常见动作：

- 每次减 5-10%
- 先减高风险或波动大部分
- 留核心底仓
- 把利润转去美债、红利、防守股、伯克希尔、港险等

### 5. 持有

长期持有的前提是：

- 买入逻辑还在
- 公司质量没有恶化
- 成本足够低
- 仓位不会影响心理安全
- 组合里有足够防守资产

他不是单纯“死拿”，而是“低成本持有 + 高位适当减仓 + 下跌再补 + 长期保留核心仓”。

## 仓位纪律

这是整套体系最重要的执行层。

1. 永远不满仓  
   不满仓就是给自己留机会。他多次提到常态仓位大约 7-7.5 成，最多也通常不超过 8-8.5 成。

2. 不轻易上杠杆  
   杠杆只适合把握极高、风险能承受的场景。普通人最好不用。

3. 现金是操作权  
   现金不是收益低的废物，而是下跌时的主动权。

4. 高收益后要转防守  
   不能让账面浮盈一直暴露在高波动资产里。赚到后要把一部分变成压舱石。

5. 不同账户有不同使命  
   进取账户负责冲锋，稳健账户负责中轴，防守账户负责活下来。

## 宏观节奏

他的宏观框架围绕美元周期、美联储、全球资金流、利率和风险偏好。

1. 加息和收水阶段  
   回笼资金，减少杠杆，等待美元资产机会。

2. 利空出尽和降息预期阶段  
   风险资产、优质科技股、美股宽基开始出现机会。

3. 经济复苏期  
   美股优先，大宗商品可提前埋伏，债券辅助。

4. 经济火热期  
   可关注大宗商品和部分消费，债券配置降低。

5. 滞胀或危机期  
   现金为王，谨慎观察债券和优质资产错杀机会。

这套宏观判断的关键不是预测绝对点位，而是提前准备现金和仓位，在周期切换时有能力行动。

## 可执行检查表

每次评估一只美股或 ETF，可以按下面流程：

1. 账户定位  
   这笔资产属于进取、稳健还是防守？

2. 资产质量  
   是否符合“第一或唯一”？是否有护城河、现金流、全球竞争力？

3. 周期位置  
   当前处于加息、降息、复苏、过热、危机中的哪一段？

4. 成本和安全边际  
   现在买入后，如果再跌 20-30%，是否还有资金和心理承受力？

5. 买入计划  
   是否设置了至少 3 个以上分批买入节点？

6. 仓位上限  
   单票、单行业、单账户的最大占比是多少？

7. 减仓计划  
   如果上涨到目标区间，是否预设 5-10% 分批减仓？

8. 资金去向  
   减仓后的钱是回现金、美债、防守股、宽基，还是等待下一次下跌？

9. 逻辑失效条件  
   什么情况说明这笔投资错了？财报恶化、竞争格局变化、监管风险、管理层问题，还是估值严重透支？

10. 复盘记录  
   每次操作必须记录：为什么买、为什么卖、成本多少、仓位多少、后续验证什么。

## 对你日后使用的建议

1. 不要试图复制历史价格节点  
   很多具体节点已经过期，博主本人也说策略会迭代。应该复制的是“节点设计方法”。

2. 先搭账户结构，再谈买卖  
   如果没有进取、稳健、防守分层，直接抄个股容易变成追涨杀跌。

3. 先把宽基和防守资产作为基础  
   对普通投资者，标普 500、纳指 100、短债/美债、防守股和现金管理，比重仓单一科技股更重要。

4. 个股只买自己能长期理解的资产  
   不懂商业模式、不懂财报、不知道竞争优势，就不要因为博主提到过而买。

5. 把“低成本/负成本”当成目标  
   真正让人能长期持有的，不是信仰，而是低成本、安全边际和已落袋的利润。

6. 用最新数据重算，而不是用旧文章直接下单  
   每次行动前，要重新看当前股价、估值、财报、利率、汇率和市场情绪。

## 最浓缩的操作口诀

- 顺势，不逆势。
- 买第一或唯一。
- 不满仓，留弹药。
- 低位分批，越跌越有计划。
- 高位减仓，利润转防守。
- 底仓长期，浮仓做 T。
- 做低成本，追求负成本。
- 进取负责进攻，稳健负责中轴，防守负责活下来。
- 策略会过期，体系要迭代。
- 投资服务生活，不让生活服务投资。
